☀️ Daily Workflow — run in this order
📰
1   Generate Fresh Briefing
Fetches news, emails, weather, markets and calendar. Generates your HTML briefing and emails it. Takes 1–3 minutes.
py briefing.py
✅
2   Review Actions → Push to To Do
Opens latest briefing in your browser with checkboxes. Tick actions for today and bulk-push to Microsoft To Do.
py briefing.py review
🗓️
3   Schedule Tasks → Push to Outlook
Proposes a schedule for your To Do tasks. Opens dashboard to review, adjust times, and push to Outlook calendar.
py outlook_scheduler.py dashboard
🔍
Preview Schedule (no browser)
Generates a proposed schedule and prints it to the log below. No Outlook events created. Quick check only.
py outlook_scheduler.py preview
⚙️ Settings & Tools
📋
Edit Scheduling Rules
Opens the rules editor in your browser. Set working hours, fortnightly cycle, task types, protected times.
scheduling_rules.html
🔑
Refresh Outlook Token
Run when Outlook auth expires (~every 90 days). Walks you through a browser login then uploads the new token to GitHub.
py outlook_email.py setup
